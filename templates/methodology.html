Methodology

This project uses dataset cleaning, feature engineering, and regression model training to predict used car resale prices.

Workflow

  1. Dataset inspection and mapping
  2. Data cleaning and validation
  3. Feature engineering and preprocessing
  4. Train/test split with stratification
  5. Train Linear Regression and Random Forest
  6. Evaluate on the same test split
  7. Save artifacts and deploy with Flask
